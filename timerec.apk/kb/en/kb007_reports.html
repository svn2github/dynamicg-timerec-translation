Time Recording

Pull reports
• Go to "Menu / Reports". Alternatively do a long click on the Day, Week or Month button
• Tap one of the reports:
- click the report name (marked red) to send out the report - also see next section
- click the "Preview" icon (marked blue) to preview a report directly on your phone without sending it out
- each report has detailed settings (tap the icon marked green)


Report settings: output and delivery
• Reports are either delivered as email attachment or uploaded to Google Drive or Dropbox.
• Report output formats:
- "HTML" is most convenient for plain viewing or simple printouts. Also note Google Cloud Print which allows to send these HTML reports directly to a printer.
- "XLS" is the Microsoft Excel spreadsheet format. Use this if you want to import your timesheet data into MS Excel or Open Office.
- "CSV" is a plain text file with comma-separated field data. This is for backwards compatibility (native XLS format has only been added with app version 5.10), new users should stick with either HTML or XLS.
- "XML" is for developers



CSV trouble shooting
• If CSV import does not work (i.e. all data imported as one cell) try changing the output format to "XLS" (see above).
