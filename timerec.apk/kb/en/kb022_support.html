Time Recording: Frequently asked questions


Does your app also run on iPhone, Windows Mobile or other platforms? Do you have plans for a web-enabled version?
No. The app only runs on Android, other versions are neither available nor planned. And yes, that's a final decision.

How do I track my holidays?
The app doesn't have extra functionality for this, but you can use tasks and notes to track and report holidays. The same applies tracking overtime hours. See these pages:
• KB003: Holidays
• KB009: Overview Delta

My hourly rate has changed. How do I put this into the app without chaning past entries?
Create a new task with the according hourly rate and assign your new entries to that.
If you've alreday used tasks before for the old entries, set that task to "inactive"

Can I use the same task with different clients?
No. A task belongs to one single client, the app has this strict hierarchy "Work unit => Task => Client".
The only workaround is to create a new task for each combination of 'common task' and client. Also see KB012: Tasks export and import.
This is a basic restriction. Right now we don't think the app will ever be able to support this. (1)

Does the app support parallel or overlapping entries?
No. No workaround available.
Note - if you want to track multiple persons or things like machinery see this page: Multi instances
This is a basic restriction. Right now we don't think the app will ever be able to support this. (1)
