Using Tasker for location based check-in and check-out in Time Recording Pro

Note Tasker is a paid app.

(A) Open Tasker, go to page "Tasks" and create actions for "Check in" and "Check out"

(B) Go to page "Profile" and create a "Location" entry

(C) Link profile with actions, note the "long click" to set the "Exit task" (i.e. the action when leaving the area)
