Time Recording

Noteworthy things for using Time Recording with your team or peers:

• For tracking multiple persons on one device: try with the Multi instances.

• Sharing your task list: Tasks Export and Import
Also note that through NFC Automation tasks can be propagated (since the task and client name are stored directly on the NFC tag).

• Changing the default file names of the Time Recording reports:
- go to the Reports screen, tap Menu/Settings.
- on Page 2 change the "Filename prefix" property, e.g. put your employee short code in there, or similar
Also note the reports can be uploaded directly to Google Drive or Dropbox (default setting is "Send as email attachment").

• If your employees forget to send in their reports on time: have them install and use the Report Reminder.

• For importing your data and settings on new devices for simplified setup:
(1) Preparation:
Pull a backup on your master device and copy the resulting "timerec.db.gz" to a web server. Or alternatively send out that backup file as email attachment to your peers.
(2) On each target device:
- Install this app: TimeRec Backup Dispatcher
- Open the "timerec.db.gz" file from step 1: either by opening the according web location (URL) in the mobile web browser, or by downloading the according email attachment using the Email app.
=> The "Backup Dispatcher" app from step 2 will now kick in and send the user to the according "Time Recording / Restore data and settings" screen for the given backup file.
